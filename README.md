 ### Resonant Debris ###


### DESCRIPTION ###
Resonant Debris(formerly(before release) - Void Salvage) is an arcade by JvVireo(me), written in Java. 
in Resonant Debris you play as a spaceship that must dodge asteroids and collect scrap. it also has difficulty. The more difficult it is, the more points you get for scrap.

### REQUIREMENTS ###
there are requrements for resonant debris:
- **JDK:** 21+
- **CPU:** 1GHz+
- **RAM:** 256mb, **but it a lot for this(it eats 150mb lol)**
- **GPU: any(including built-in in CPU)**
- **OS:** Windows 7 Update 1( but if not the Oracle, it can be started on Windows XP)
- **Hard disk drive space**:500 + 32 = 532mb
